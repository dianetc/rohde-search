# Rohde Search

A searchable archive of the [Ali Rohde Jobs](https://alirohdejobs.substack.com/) newsletter - Chief of Staff, BizOps, and VC roles in tech. This doesn't include jobs (since many are likely expired) but instead includes interesting companies and their respective description, location, and industry. 

Data source: [Ali Rohde Jobs](https://alirohdejobs.substack.com/) by [Ali Rohde](https://twitter.com/alirohde)

## Error Fixes

If a company description is incorrect or you wish to update any other information, please open an issue.
